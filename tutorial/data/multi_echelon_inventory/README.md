# Multi-Echelon Inventory Data

These CSV files are copied from the `simpy_3.0/data` directory of:

https://github.com/anshul-musing/multi-echelon-inventory-optimization

Files:

- `demandData.csv`: historical demand observations for stocking nodes 1-5.
- `leadTimeExtraDays.csv`: historical extra lead-time day observations.
